3D Chart allows you to style the background in the following ways:
  • backgroundColor - static background color
  • backgroundGradient - static background gradient set as an array of "offset, color" strings
  • backgroundImage - static background image
  • backgroundTexture - dynamic background image, which rotates with the camera. It is set as an array of 6 images, one for each side of a 3D cube
