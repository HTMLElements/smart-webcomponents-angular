Filter with Set Filter

Set Filters Demo

This demo illustrates how to create set filters, which allow users to filter data based on a predefined set of values. Set filters provide a user-friendly interface for selecting multiple values from a dropdown list, making it easier to apply complex filtering conditions.

Key Benefits of Set Filters

  • Select multiple values from a predefined list
  • Improve usability for categorical data
  • Quickly narrow down grid data
  • Avoid manual entry of filter conditions

Set filters are particularly useful when working with categorical data or when users need to filter based on specific criteria without manually entering filter expressions.

Customizing the Filter Editor

The demo also demonstrates how to customize the filter editor by providing a custom implementation of the filter editor’s onChange event. This gives developers greater control over the filtering behavior.

Why Customize onChange?

  • Update other components dynamically
  • Trigger API calls
  • Apply additional filters
  • Implement custom filtering logic

Review Date Column Example

The 'Review Date' column includes a custom set filter editor with predefined options:

  • Last Week
  • Last Month
  • Last Year

When a user selects one of these options, the onChange event is triggered. The application then applies the corresponding date filter to the grid data.

This approach provides a more intuitive and efficient filtering experience, especially when working with date-related or categorical data that benefits from predefined filter options.
