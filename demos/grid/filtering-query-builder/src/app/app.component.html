Filter with Query Builder

This demo illustrates how to create a custom filtering side panel for the Grid. The side panel contains a query builder component that allows users to construct complex filter conditions using a user-friendly interface. The query builder supports various logical operators (AND, OR) and comparison operators (equals, not equals, greater than, less than, etc.) to enable flexible filtering of the grid data. Users can easily add multiple filter rules and groups to refine their search criteria, making it easier to find specific data within the grid
